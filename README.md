# IP Heimdall 👁️

**Определитель IP-адресов и геолокации**

Современное веб-приложение для определения информации об IP-адресах с красивым интерфейсом и подробными данными о геолокации, сетевых параметрах и безопасности.

---

## ⚡ Возможности

✅ **Определение IP-адреса**
- Поиск информации по введённому IP-адресу
- Быстрое определение собственного IP-адреса
- Валидация формата IP (IPv4 и IPv6)

✅ **Полная информация**
- Геолокация: страна, город, регион, континент
- Координаты: широта и долгота
- Часовой пояс и текущее время
- Сетевые данные: ASN, ISP, организация, домен

✅ **Безопасность**
- Определение VPN, Proxy, Tor
- Выявление датацентров и ботов
- Информация об истории злоупотреблений

✅ **Современный дизайн**
- Тёмная тема с неоновым синим
- Плавные анимации и переходы
- Полностью адаптивный интерфейс (мобильные устройства)
- Интуитивный и удобный UX

## 🔌 Используемые API

Приложение использует несколько бесплатных API для отказоустойчивости и получения полной информации:

1. **api.ipapi.is**
   - Основной источник геолокации и сетевых данных
   - Расширенные сведения о безопасности (VPN, Proxy, Tor)
   - Информация о типе подключения
   - Данные об организации и абузе

2. **ipwho.is**
   - Резервный источник геолокации и сетевых данных
   - Используется как fallback при ошибках/лимитах основного API

## 💻 Использование

### Определить собственный IP

1. Нажми кнопку **"Мой IP"**
2. Приложение автоматически загрузит информацию о твоём IP

### Поиск по IP-адресу

1. Введи IP-адрес в поле ввода
2. Нажми **"Найти"** или нажми **Enter**
3. Получи полную информацию об IP

### Форматы IP

Поддерживаются оба формата:
- **IPv4**: `91.224.74.173`
- **IPv6**: `2001:0db8:85a3:0000:0000:8a2e:0370:7334`
